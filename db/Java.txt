Java


   Die term Java kan verwys na die volgende:

   In geografie:
     * Java (eiland), Indonesië, die eiland met die hoogste bevolking in
       die wêreld
          + Javaanse taal, taal wat gepraat word op die eiland Java
     * Java koffie, 'n tipe koffie plant wat se oorsprong die eiland Java
       is, of ook 'n slang woord vir koffie
     * Java, Georgia, 'n dorp in die Republiek van Georgia
     * Java, New York, VSA
     * Java, Suid Dakota, VSA; met poskode 57452
     * Java, Virginia, 'n dorp in Pittsylvania, Virginia, VSA; met poskode
       24565

   In rekenaar wetenskap:
     * Java (Sun), 'n tegnologie wat ontwikkel is deur Sun Microsystems,
       wat insluit :
          + Java (programmeertaal), 'n objek georienteerde hoë-vlak
            programmeertaal
          + Java virtual machine, die java virtuele masjien wat java
            bit-kode uitvoer
          + Java platform
               o Java Platform, Standard Edition
               o Java Platform, Enterprise Edition
               o Java Platform, Micro Edition
     * JavaScript

   Java kan ook beteken:
     * Java (band), 'n Franse musiekgroep
     * Java (bordspel), 'n bordspel gebaseer op die eiland Java
     * Java (cachaça), 'n tipe cachaça
     * Java (dans), 'n Paryse Bal-musette dans
     * Java Man, een van die eerste voorbeelde van Homo Erectus wat ondek
       is
     * "Java" (lied), 'n lied deur Allen Toussaint. 'n Gewilde weergawe is
       ook opgeneem deur Al Hirt, wat later gebruik is in The Muppet Show
     * HMS Java, drie skepe van die Britse vloot
