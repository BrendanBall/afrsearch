Oracle Corporation


   Die woord Oracle kan verwys na die Amerikaanse maatskapy Oracle
   Korporasie, of die Oracle databasis sagteware wat hulle skryf en
   versprei.

   Inhoud

        * 1 Die maatskapy
        * 2 Die Oracle databasis
        * 3 Ander produkte
        * 4 Eksterne skakels

[wysig] Die maatskapy

   Oracle se hoofkantoor is geleë in Redwood Shores, Kalifornië, VSA.
   Hulle aandele word op die New Yorkse aandelebeurs genoteer.

[wysig] Die Oracle databasis

   Oracle word beskou as een van die mees populêre
   databasis-beheerstelsels ter wêreld. Die Oracle databasis is beskikbaar
   op meeste bedryfstelsels insluitend Microsoft Windows, Linux en Unix.

[wysig] Ander produkte

   Oracle verskaf talle ander sagteware produkte, onder andere:
     * Oracle Applications (Industrie applikasies)
     * Oracle Colaboration Suite (sluit in e-pos, telefonie, ens.)
     * Ontwikkelingspakete (Oracle Forms, Reports, ens.)
